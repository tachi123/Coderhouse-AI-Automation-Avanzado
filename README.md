## 🛠️ Cómo Importar las Prácticas (.json)

Para utilizar los archivos de práctica en tu propia instancia de n8n, sigue estos pasos:

1. **Descarga el archivo JSON**: Navega a la carpeta de la clase correspondiente y descarga el archivo `.json` del workflow.
2. **Entra a n8n**: Abre tu instancia de n8n en el navegador.
3. **Crear nuevo Workflow**: Haz clic en el botón de añadir un nuevo workflow.
4. **Importar**:
   - Puedes arrastrar el archivo JSON directamente al lienzo de n8n.
   - O ve al menú (tres puntos en la esquina superior derecha) -> **Import from File**.
5. **Configura tus Credenciales**: Recuerda que deberás configurar tus propias APIs (OpenAI, Groq, Google Gemini, etc.) en los nodos correspondientes para que los flujos funcionen.
