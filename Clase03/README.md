## Práctica: Implementación de RAG con n8n
Los flujos de trabajo incluidos demuestran el proceso de:
1. **Carga de Documentos**: Uso de cargadores de datos.
2. **Fragmentación (Chunking)**: Uso de Recursive Character Text Splitter.
3. **Embeddings**: Integración con Google Gemini.
4. **Almacenamiento**: Uso de Vector Stores simples para recuperación de información.
